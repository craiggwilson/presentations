Questions to Ask Yourself

  • Will your application be read or write intensive?
  • How do you anticipate your documents changing over time?
  • Will your documents grow over time?
  • Do you anticipate needing to shard your data?

Schema - Book


book = {
  _id: "ObjectId",
  title: "string",
  *author: "ObjectId",
  isbn: "string",
  publisher: {
    name: "string",
    date: "timestampe",
    city: "string"
  },
  available: "boolean",
  pages: "int",
  summary: "string",
  subjects: ["string"]
  notes: [{
  	*user_id: "ObjectId", 
  	note: "string"
  }],
  language: "string"
}

Schema - Author

	_id: "ObjectId",
	first_name: "string",
	last_name: "string"
}

Schema - User
